JDK Dynamic Proxy

Posted on 2019-05-08

Abstract

代理是一种设计模式。当我们想增加或者修改一个已经存在类中的方法，我们可以通过创建并使用代理对象来实现。通常，代理对象具有与原始对象相同的方法，并且在Java代理类中通常会扩展原始类。代理有一个原始对象的句柄，可以调用该对象的方法。
